## [0.1.0] - 2024-11-16

### Added
- 🎉 초기 릴리즈
- ✅ Button 컴포넌트
  - 3가지 variant (solid, outline, ghost)
  - 6가지 intent 색상
  - 3가지 size
- ✅ Checkbox 컴포넌트
  - 체크마크 애니메이션
  - 접근성 지원
- ✅ Switch 컴포넌트
  - 부드러운 토글 애니메이션
  - Spring 효과
- ✅ TextField 컴포넌트
  - Outline/Filled variant
  - 에러/성공 상태
  - Helper text 지원
- ✅ Dialog 컴포넌트
  - Portal 렌더링
  - ESC/Overlay 닫기
  - 애니메이션 지원

### Design System
- 🎨 디자인 토큰 시스템 구축
  - Colors (Light/Dark 테마)
  - Typography
  - Spacing
  - Radius
  - Shadow
- 🎨 Vanilla Extract CSS-in-JS
- 🎨 Recipe 패턴 적용

### Documentation
- 📚 Storybook 8 설정
- 📚 각 컴포넌트별 10+ Stories
- 📚 접근성(A11y) 테스트 추가
